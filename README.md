## 快速使用

#### 全局安装🔥
~~~
npm install -g rs-admin-cli
~~~

#### 新建项目
~~~
rs init yourProjectName
~~~

#### 运行
~~~
cd yourProjectName
npm start
~~~

#### 打包
~~~
npm run build
~~~
